# Workshop Documentation

This directory contains the workshop tasks and guides for learning container technologies and Azure deployments.

## Tasks

Complete these tasks in order to progressively build your container and Azure deployment skills:

1. **[Local Docker Setup](01-local-docker-setup.md)** - Build and run containers locally with Docker
2. **[Azure App Service Deployment](02-app-service-deployment.md)** - Deploy containers to Azure App Service
3. **[Azure Container Apps Deployment](03-container-apps-deployment.md)** - Deploy to serverless Container Apps
4. **[Azure Kubernetes Service Deployment](04-aks-deployment.md)** - Deploy to AKS for production orchestration

## Time Commitment

Each task takes approximately **30 minutes** to complete.

## Prerequisites

- Docker Desktop or Rancher Desktop
- PowerShell Core
- Azure CLI
- .NET 9 SDK
- kubectl (for Task 4)

Run `.\check-prerequisites.ps1` from the repository root to verify your setup.
